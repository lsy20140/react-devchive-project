## ✨ 프로젝트 소개

***Devchive** = **Dev**(elopment) + (Ar)**chivce***


#### 🤔 어떤 형식으로 작성할까 고민돼요

개발 공부 중 기록은 남기고 싶지만 텅텅 비어있는 입력란을 채우는 것이 막막하고 부담스러우셨나요?

꼭 필요한 입력 항목으로만 구성된 템플릿을 통해 작성 형식에 대한 고민 없이 빠르고 간편하게 기록을 남길 수 있는 서비스입니다.
